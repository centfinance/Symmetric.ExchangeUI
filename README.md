## Development

Create `.env` file to change environment variables. Default values are:

```
APP_CHAIN_ID=1
APP_GAS_PRICE=100000000000
```

Run & build:

```
npm install
npm run serve
npm run build
```

### Supported Networks
```
Ethereum main network
Gnosis
Celo
Kovan
POA
Catalyst (coming soon)
Cent public network (coming soon)
```
